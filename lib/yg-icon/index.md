## API

### `yg-icon`

| Name    | Type           | Default  | Summary |
| ------- | ------------- | ----- | ----- |
| ygSpin | `boolean` | `false` | Rotate icon with animation |
| ygType | `string` | - | Type of ant design icon |

## DEMO

```html
<yg-icon nzType="loading" nzSpin></yg-icon>
<yg-icon nzType="user"></yg-icon>
```
